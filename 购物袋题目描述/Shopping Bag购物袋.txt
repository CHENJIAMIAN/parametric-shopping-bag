# 参数化建模测试题 ：购物袋 Shopping Bag

## 目标

请根据提供的购物袋刀线图和生成规则，制作一个可运行的参数化 3D 购物袋模型。

模型需要支持用户自定义尺寸，并允许用户在包装表面放置图片素材。尺寸变化后，素材不能被拉伸、压扁或明显错位。

## 输入资料

我们会提供：

1. 购物袋刀线图
2. 刀线图生成规则说明
3. 测试用图片素材
4. 三组测试尺寸

## 功能要求

1. 支持自定义尺寸：

   * Width
   * Height
   * Gusset

2. 支持至少三组尺寸预设：

   * Small
   * Medium
   * Large

3. 支持 3D 成型效果展示。

4. 至少展示以下结构：

   * Front panel
   * Back panel
   * Side gusset
   * Bottom structure
   * Handle

5. 支持将图片 / logo / 图案放置到包装表面。

6. 尺寸变化后，表面素材需要保持正确比例，不得明显变形。

7. 模型需要可在网页中运行并交互查看。

## 交付物

请提交：

1. 可运行 demo
2. 源码
3. README
4. 简短说明：

   * 如何修改尺寸
   * 如何替换素材
   * 建模思路
   * 当前限制

## 验收重点

我们会重点检查：

1. 尺寸变化后购物袋比例是否自然
2. 侧风琴和底部结构是否合理
3. 表面素材是否变形
4. 正面、背面、侧面是否对应正确
5. 代码是否清晰、可维护
